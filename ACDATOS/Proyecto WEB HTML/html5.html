Estructura básica de una página web:


Los archivos con extensión .html deben contener la siguiente estructura:

<!DOCTYPE html> ----(“Definimos el tipo de documento”)
<html lang= “es”>-------(“Definimos el lenguaje = “Español” ”)
<head>--------------------(“Indica la cabecera del documento”)
<meta charset=”utf-8”/ >(“Se define la codificación usada”)
<title > </title > ------------(“Especifica el título del documento”)
  </head>
   <body>----------(“Es el cuerpo del html, en él se encuentra el contenido de la pag”)
   <header> -----------(“Tipo de cabecera para añadir, nombre, slogan o logo”)
    <nav>------------(“Barra o menú de navegación, dentro puede haber listas <ul > <li >para dar forma”)
    </nav >
   </header >
<section> -----------(“Divide el contenido en secciones”)
    <article> </article>---(“Caja de contenido que puede estar dentro o no del section”)
</section>
<aside ></aside> --------------(“Divide el cuerpo en 2 columnas”)
<div></div> -----------------(“Similar al section, pero a este se le puede dar formato css”)
<footer></footer> -------------(“Es el pié de pag”)
</body>
</html>

Extensiones de archivos:


Archivos con extensión .html:

Html es una estructura de hipertexto “sistema de organización y presentación de datos basado en la unión de fragmentos gráficos o escritos.” derivado del SGML.

Lo que hacen los navegadores que utilizan los usuarios, como Mozilla Firefox, Google Chrome o Internet Explorer, es interpretar este contenido desarrollado en HTML y exhibirlo de manera tal que las personas puedan interpretarlo (como texto, imágenes, etc.).

Archivos con extensión .htm:

Htm es una abreviatura de Html que se abrevia para la compatilidad con sistemas operativos más antiguos.

Archivos con extensión .xhtml:

El XHTML es un lenguaje que adapta el HTML al XML.

Vuelve compatibles ambos lenguajes (el HTML y el XML).

De esta forma, mantiene la mayoría de las características del HTML, aunque con elementos del XML.
El XHTML, por lo tanto, otorga mayor robustez y capacidad de adaptación.

Etiquetas de bloque


Un elemento de bloque es aquel que forma un “bloque” y se posicionan de forma vertical con un nuevo salto de línea, puede contener elementos de tipo línea.

La altura y anchura cambia en base al contenido y se le pueden aplicar por css.

Etiquetas de línea


Estos elementos se posicionan horizontalmente en línea con otros elementos, la altura y anchura se definen en base al contenido y no por css.

Algunos elementos que corresponden con el tipo en línea:

< a href="#"> ----- Se usa para definir un destino o un origen de un enlace.
Moodle Ies Virgen del Carmen

< br> ----- Se utiliza para dar un salto de línea.

< span> ------ Añade un estilo determinado dentro de una sección del texto.
Color: Rojo,verde, azul.

< img src> "nombreImagen.jpg" -------- Añade una imagen, se le pueden añadir atributos.
< i>----- Inclina el texto --- "Texto inclinado".
< b> --- Pone el texto en negrita --- Texto en negrita.
< u> --- Subraya el texto --- Texto Subrayado.
< s> --- Tacha el texto --- Texto tachado.
< Font> --- Formatea el tipo y tamaño del texto en algunos puntos del documento específicos.
Tipo para formatear

< strong> --- Pone el texto en negrita, como la etiqueta < b > .
< var> --- Nos indica que el contenido de la misma es una variable.
< input> --- Se utiliza para ingresar datos, usado frecuentemente en un formulario.
  Ejemplo:

< select> --- Se utiliza para crear listas despegables.
< button> --- Similar al input: Se utliza para crear botones, permite más opciones.

Estilos


Los estilos son extensiones .css que tienen definidos el formato (fuente, color de fuente, fondo...)

Se pueden aplicar de tres formas diferentes:

Mediante un archivo externo: "Creamos un archivo.css le ponemos el formato. Ejemplo:

< html xmlns="http://www.w3.org/1999/xhtml">
    < head>
< meta http-equiv="Content-Type" content="text/html; charset=iso-8859-1" />
< title>Ejemplo de estilos CSS en un archivo externo< /title>
< link rel="stylesheet" type="text/css" href="Prueba.css" media="screen" />
    < /head>
                                
< body>

< /body>
< /html>
