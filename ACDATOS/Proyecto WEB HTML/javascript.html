Historia JavaScript

A principios de los años 90, el acceso a Internet se hacia con módems a una velocidad máxima de 28.8 kbps.


Empezaron a desarrollarse aplicaciones web cada vez más complejas y una velocidad de navegación tan lenta, surgió la necesidad de un lenguaje de programación que se ejecutara en el navegador del usuario. De esta forma, si el usuario no rellenaba correctamente un formulario, no se le hacía esperar mucho tiempo hasta que el servidor volviera a mostrar el formulario indicando los errores existentes.


Brendan Eich, un programador que trabajaba en Netscape, pensó que podría solucionar este problema adaptando otras tecnologías existentes (como ScriptEase) al navegador Netscape Navigator 2.0, que iba a lanzarse en 1995. Inicialmente, Eich denominó a su lenguaje LiveScript.

Posteriormente, Netscape firmó una alianza con Sun Microsystems para el desarrollo del nuevo lenguaje de programación.Además, justo antes del lanzamiento Netscape decidió cambiar el nombre por el de JavaScript. La razón del cambio de nombre fue exclusivamente por marketing, ya que Java era la palabra de moda en el mundo informático y de Internet de la época.

Variables

El lenguaje JavaScript utiliza la palabra reservada 'var' para crear variables, una vez declarado una variable por ejemplo 'var numero = 0', sí posteriormente queremos volver a utilizar la variable con indicar solamente el nombre es suficiente. Podemos definir variable declarada y no declarada, es decir, una variable es declarada cuando se le da un valor como en el ejemplo anterior, y no declara si no se le asigna a la variable un valor.


Las variables las podemos encontrar en cualquier lenguaje de programación (java, c++ , etc) la finalidad de una variable consiste en asignar un nombre (identificador) a un determinado valor. El identificador tiene que cumplir unos requisitos, en el caso de JavaScript son las siguientes:


- Sólo admite letras minúsculas, mayúsculas, números o el símbolo _ (guion bajo)

- Debe empezar con letras o el guion bajo

- Como máximo se pueden utilizar 31 caracteres

- Se pueden utilizar letras Unicode (es válido el identificador año


Existen varios tipos de variables (numericos, cadenas de texto, booleanos, arrays) que veremos a continuación:

Numéricos


La variables de tipo numérico se utilizan para almacenar numeros existen los de tipo entero(integer) y los de tipo decimal(float), las variables de tipo decimal utiliza el carácter “.” Para separar la parte entera de la decimal.


Ejemplo


var iva = 12; // variable tipo entero

var total = 12.65; // variable tipo decimal


Cadenas de texto

A diferencia de la variable numerica, la variable cadenas de texto se utilizan para almacenar caracteres, palabras y/o frases de texto. Una forma de identificar si es un cadena de texto es viendo si la variable está definida entre comillas dobles o simples, indican su comienzo y su final.


Ejemplo


var mensaje = "Bienvenido a nuestro sitio web";

var nombreProducto = 'Producto ABC';

var letraSeleccionada = 'c';


El objeto String() de JavaScript dispone de varios métodos y propiedades que nos permitirán realizar operaciones útiles con cadenas, que a continuación te describimos:


Convertir mayúsculas y minúsculas (Los métodos toLowerCase() y toUpperCase() devuelven una cadena convertida a minúsculas y mayúsculas, respectivamente).


Obtener la longitud de una cadena(Con la propiedad length obtendremos la longitud de una cadena (espacios incluidos)).


Unir o concatenar cadenas (Normalmente podemos unir cadenas (concatenarlas, es decir, unir) usando el operador ' + ') del siguiente modo:


Ejemplo


var mensaje = "Bienvenido";

var mensaje2 = "a este tutorial";

var resultado = (mensaje + mensaje2);


Convertir a cadena de caracteres (Podemos convertir un número a cadena concatenando un número con una cadena usando el operador '+', o bien con la función String().


Booleanos

Las variables de tipo boolean o booleano también se conocen con el nombre de variables de tipo lógico, tienen la misma funcionalidad que en el lenguaje de programación C++.


Una variable de tipo lógico o booleano no sirve para almacenar números, cadenas de caracteres, etc, solo sirve para almacenar una valor lógico: True(Verdadero) o False (Falso):

var clienteRegistrado = false;

var ivaIncluido = true;

Arrays

Al igual que en muchos lenguajes de programación (los arrays en C++ son identificados como arreglos, vectores o matrices, como podemos ver en JavaScript tambíen son identificados con ese mismo fin), pero el término mas utilizado es array o vector.


En un array su contenido puede ser del mismo tipo o diferente. Lo podemos ver con un ejemplo


Ejemplo

Si un profesor necesita almacenar las notas del primer examen de los 10 alumnos de la clase, el array sería de la siguiente manera(pero este código es poco eficiente, pero no incorrecto):

var nota1 = 5;

var nota2 = 8;

...

var nota10 = 7.3;


¿Por que decimos que el código anterior es poco eficiente?


La explicación de que es poco eficiente es debido a que si queremos saber el nombre de toda la población de Jaén tendríamos que crear una gran cantidad de variables.

Entonces lo correcto sería crear una variable que almacenara toda la información que queremos.

var nombres = ["Jesús", "Carlos", "Pepito", "Dani", "Jose"...];

Ahora, una única variable llamada nombres almacena todos los valores relacionados entre sí, en este caso los nombres de una población.

La definición de un array se realiza entre corchetes([]) y para indicar su comienzo y su final se utiliza el carácter , (coma) para separar los elementos del interior del array:

var nombre_array = [valor1, valor2, ..., valorN];

Por defecto en programación las posiciones del vector empiezan en (0...n), y esta característica de la programación al comienzo de programar es propenso de un error, pero una vez definido el vector es muy fácil acceder a la posición que que queremos ver un contenido.

var diaSeleccionado = dias[0]; // diaSeleccionado = "Lunes"

var otroDia = dias[5]; // otroDia = "Sábado"

Estructuras de Control

Las estructuras de control se utilizan para aplicar unas funciones u otras según las condiciones que nosotros queramos establecer, si no fuera por esto solo podríamos ejecutar una instrucción después de otra.

Para realizar este tipo de programas son necesarias las estructuras de control de flujo, que son instrucciones del tipo "si se cumple esta condición, hazlo; si no se cumple, haz esto otro". También existen instrucciones del tipo "repite esto mientras se cumpla esta condición".

Cuando un programa lleva esta estructura deja de ser sucesión lineal y pasa a llamarse programa “inteligente”.

Permiten controlar el flujo de un programa, tomar decisiones dependiendo del estado de una variable, repetir instrucciones o acciones un número determinado de veces, etc. En general las estructuras de control son 3:

  1. Estructuras condicionales
  2. Estructuras de Repetición

Estructuras Condicionales

Estructura if

Se emplea para tomar decisiones en función de una condición. Su definición formal es:

if(condicion) {

...

}

La funcionalidad de está estructura de control es la siguiente, si la condición que hay en el interior del paréntesis se cumples , es decir, la condición es (True) se realiza la ejecución de la estructura del paréntesis lo que hay dentro del los ({}), una vez terminada la condición sigue con la ejecución del resto del script, en caso de que la condición sea (False), se salta la estructura de control y continua la ejecución del script.

Ejemplo

var mostrarMensaje = true;

if(mostrarMensaje) {

alert("Hola Mundo");

}

En el ejemplo anterior, el mensaje sí que se muestra al usuario ya que la variable mostrarMensaje tiene un valor de true y por tanto, el programa entra dentro del bloque de instrucciones del if.

El ejemplo se podría reescribir también como:

var mostrarMensaje = true;

if(mostrarMensaje == true) {

alert("Hola Mundo");

}

En este caso, la condición es una comparación entre el valor de la variable mostrarMensaje y el valor true. Como los dos valores coinciden, la igualdad se cumple y por tanto la condición es cierta, su valor es true y se ejecutan las instrucciones contenidas en ese bloque del if.

En la programación solemos cometer un error muy frecuente, cuando utilizamos el comparador ==, muchas veces tendemos a escribir con un solo =, y esto es un grave error, porque lo que estamos realizando entonces es asignar un valor.

Los operadores AND y OR permiten encadenar varias condiciones simples para construir condiciones complejas:

Ejemplo

var numero1 = 3;

var numero2 = 12;

if (numero1 == 3 && numero2 == 12){

...

}

La condición anterior está formada por una operación AND sobre dos variables. Lo que realiza dicha condición, si se cumple que la variable numero1 es 3 y la variable numero2 es 12 entonces entra a la condición, en caso de que solamente una variable no tenga asignado ese valor la condición no se puede realizar.

Estructuras de Repetición

Estructura For

La estructura for es una condicón un poco mas compleja que la del if, nos permite realizar repeticiones (también es conocida como bucle).

Ejemplo

for(inicializacion; condicion; actualizacion) {

...

}

El funcionamiento del bucle for se estructura de la siguiente manera: mientras la condición indicada se siga cumpliendo, se repite las instrucciones que están dentro del bloque for, por cada ejecución de bloque terminada, las variables utilizadas en la condición del for se actualizan.

  1. La "inicialización" es el valor que utilizamos para indicar en que posición empieza el bucle.
  2. La "condición" es el único elemento que decide si continua o se detiene la repetición.
  3. La "actualización" es el nuevo valor que se asigna después de cada repetición a las variables que controlan la repetición.

Ejemplo

var mensaje = "Hola, estoy dentro de un bucle";

for(var i = 0; i < 5; i++) {

alert(mensaje);

}

Lo que realiza este ejemplo es lo siguiente: creamos una variable llamada mensaje, y cuando entramos al for se imprimira el mensaje de la variable 5 veces ya que es lo que el bucle indica.

El bucle se inicializa a 0 de la siguiente manera: var i = 0;

La condición es que mientras la variable i sea menor que 5 el bucle se sigue repitiendo: i <5;

La actualización i++ indica que por cada vuelta que el bucle realice i se incrementa en 1, al inicio i=0, una vez que ha imprimido el mensaje, i se actualiza y entonces i = 1, ¡OJO!(El bucle realiza 5 repeticiones pero se para en el valor 4 en vez de 5, como hemos dicho previamente antes en programación siempre se suele empezar por 0 en vez de por 1);

Ejemplo

var nombres = ["Jesús", "Daniel", "Jose", "Ana", "Juanma", "Pepe", "Cristina"];

for(var i=0; i <7; i++) {

alert(nombre[i]);

}

El ejemplo anterior se efectua de la siguiente manera, creamos un array de 7 elementos que empiezan en la posición 0(“Jesús”) y llega hasta la posición 6(“Cristina”), y el bucle se inicializa a 0 y llega hasta que i sea menor que 7, y la ejecución del bucle es la impresión de lo que contiene el array.

alert(nombre[i]); (Significa que nos muestra lo que hay en la posición del array i vale 0 en la primera vuelta del bucle por lo que mostrara(Jesús) “nombre[0] = Jesús” en la salida, luego i se incrementa y vale 1 entonces en la salida imprimira (Dani)).

Estructura for..in

Es una estructura simplificada para recorrer un Array, utilizando una variable como contador del índice del Array.

for (contador in Array){

alert (Array[contador]);

}

Aquí, el alert() mostrará todos los elementos del Array, tal como se ve en el siguiente ejemplo:

Ejemplo

var dias = ["Lunes","Martes","Miércoles","Jueves","Viernes","Sábado","Domingo"];

for(i in dias) {

alert(dias[i]);

}

Estructura while

La estructura de repetición while se utiliza cuando no se sabe cuantas veces se tiene que repetir un bucle, pero si sabemos cuando tiene que parar mediante una condición.

while (condición){

... // instrucciones a repetir

}

Ejemplo

var numero = 0;

while (numero <3) {

alert('mensaje');

numero++;

}

En este ejemplo al incio del while numero vale 0 y va incrementandose con la sentencia numero++ de 1 en 1 cada vez que se cumple la condición.

Estructura do...while

Es similar a la anterior, la diferencia está en que esta última, evalúa la condición al final del bucle, es decir que por lo menos una vez se ejecutará el bucle.

do {

... // instrucciones a repetir

} while (condición)


Ejemplo

var numero = 0;

do {

alert('mensaje');

numero = 3;

} while (numero <3);

En este ejemplo la variable numero vale 0 y al entradar a la condicion de do...while porque como sabemos se realiza minimo una vez al llegar a la condición el número ha sido cambiado a 3 por lo tanto no se vuelve a repetir.
