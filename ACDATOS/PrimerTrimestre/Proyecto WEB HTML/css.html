Tutorial CSS3

CSS es un lenguaje de programación que acompaña al HTML, se encarga de dar formato al código HTML de la página o páginas. El CSS se puede declarar de tres maneras diferentes: Dentro de una etiqueta, en el head y en una hoja de estilos externa

Desde etiqueta:

<html>
	<head>
	</head>
	<body>
	<font color="red">Este texto tiene otro color</font>
	</body>
	</html>

Desde head 

<!DOCTYPE html>
	<html lang="es">
	<head>
	<title> Ejemplo css </title>
	<style>
	#color {
	color: red;
	}
	</style>
	</head>
	<body>
	<h1> Ejemplo desde etiqueta</h1>
	<p id= "color">Este texto tiene otro color</p>

	</body>
	</html>

Desde hoja de estilos externa 

<!DOCTYPE html>
	<html lang="en">
	<head>
	<LINK href="estilocss.css" rel="stylesheet" type="text/css">
	<title> Ejemplo css </title>
	</head>
	<body>
	<h1> Ejemplo desde etiqueta</h1>
	<p id= "color">Este texto tiene otro color</p>
	
	</body>
	</html>
